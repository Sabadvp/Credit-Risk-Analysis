# Credit Risk Analysis using Machine Learning

## Project Overview

This project focuses on predicting the likelihood of loan default using the **Home Credit Default Risk dataset**. The objective is to develop and compare multiple machine learning models to identify high-risk applicants and support better credit decision-making.

The analysis includes data preprocessing, feature engineering, model development, evaluation, and interpretation of results.

---

## Dataset

* **Source:** Home Credit Default Risk (Kaggle)
* **Link:** https://www.kaggle.com/competitions/home-credit-default-risk/data
* The dataset contains financial, demographic, and behavioral information about loan applicants.

> Note: Due to size and licensing constraints, the dataset is not included in this repository. A link is provided instead.
